# Plan de Table

Ce projet contient une page HTML interactive permettant de gérer un plan de table.

Ouvrez `index.html` dans un navigateur pour déplacer les tables, modifier l'ordre des invités et désormais sauvegarder ou imprimer l'agencement.

## Fonctionnalités

- **Déplacement des tables** : glissez-déposez pour ajuster la disposition.
- **Réorganisation des invités** : modifiez l'ordre des convives directement dans le panneau latéral.
- **Sauvegarde** : cliquez sur **Enregistrer** pour stocker la configuration dans le navigateur (localStorage). La disposition sera rechargée automatiquement lors de la prochaine ouverture de la page.
- **Impression** : utilisez **Imprimer** pour générer une version imprimable du plan.

## Déploiement rapide

Publiez simplement ce dossier sur GitHub et activez GitHub Pages pour partager le plan de table en ligne.
